Plex Source for Lampa — install notes

1. Host plugin.js at an HTTP/HTTPS URL reachable by the device running Lampa.

Recommended URL:
https://cdn.jsdelivr.net/gh/boiler4/lampa-plex-source@main/plugin.js

Alternative raw GitHub URL:
https://raw.githubusercontent.com/boiler4/lampa-plex-source/main/plugin.js

2. In Lampa:
- Settings
- Plugins / Extensions
- Add plugin by URL
- Paste the plugin.js URL
- Reload Lampa if needed

3. Configure:
- Settings → Plex Source
- Plex server URL, e.g. http://192.168.1.10:32400
- Plex token
- Check connection

Do not publish your Plex token.

Playback note: on clients where Lampa lets you choose a player, Plex Source uses the player configured for torrent/file playback.
