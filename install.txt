Plex Source for Lampa — install notes

1. Host plugin.js at an HTTP/HTTPS URL reachable by the device running Lampa.

Example after publishing to GitHub:
https://raw.githubusercontent.com/boiler4/lampa-plex-source/main/plugin.js

2. In Lampa:
- Settings
- Plugins / Extensions
- Add plugin by URL
- Paste the plugin.js URL
- Reload Lampa if needed

3. Configure:
- Settings → Plex Source
- Plex server URL, e.g. http://192.168.1.10:32400
- Plex token
- Check connection

Do not publish your Plex token.
